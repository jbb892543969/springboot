# 说明
springboot+vue学习demo

前台利用vue2.0+elementUI 搭建后台管理系统

后台使用springboot+springSecurity进行鉴权与认证，querydsl简化jpa操作

运行方式: 后台配置数据库与redis 开启项目，前台npm install后npm run dev 运行 账号admin，密码123456
